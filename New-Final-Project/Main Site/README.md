# Dine Finder

IST 256 Final Project. Group 5: Will Cressman, Logan Henry, Jeremy Lucotch, Mason Miller.

Restaurant browser built in Angular 18 with Bootstrap 3 styling. Data lives in a local JSON file and in localStorage.

## Setup

Install Node.js (LTS) from https://nodejs.org/ and the Angular CLI:

```
npm install -g @angular/cli
```

## Run

From inside the Main Site folder:

```
npm install
npm start
```

Then open http://localhost:4200.

## Build

```
npm run build
```

Output goes to `dist/dine-finder/browser/`.

`node_modules`, `.angular`, and `dist` are auto-generated by npm and the Angular CLI. They are not part of the source and should not be in the submission zip.
